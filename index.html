FlashBrain AI

Introducing . . . FlashBrain AI! Using Javascript, Next.JS, React.JS, and Material-UI for the primary development, I created a website with OpenAI API integration to create flashcards for the user based off a given word / prompt. Then, 10 flashcards will be automatically created based on the users input. I also used Firebase for the database to store everything per user, as well as Clerk Authentication, to verify that each user was able to save their own flashcard sets under their own account. Furthermore, I incorporated Stripe to work with payments in case users want to upgrade from a basic to a pro account to access more features.

* Successfully deployed to 200+ visitors, 30+ sign-ups on day one.

VALO-SCOUT

Passionate about VALORANT? VALO-Scout is your ultimate companion. Whether you're eager to stay updated on top streamers from elite teams like FNATIC or looking for agent recommendations tailored to the users playstyle, VALO-Scout has you covered. My innovative app uses Retrieval-Augmented Generation (RAG) technology to enhance the users gaming experience. With two specialized Pinecone databases—one dedicated to tracking VALORANT streamers and another focused on agent strategies—VALO-Scout offers unparalleled insights. The custom-built OpenAI bot, "Brimstone," dynamically retrieves and analyzes data from these databases, delivering precise information and personalized recommendations. Elevate your VALORANT gameplay with VALO-Scout, the tool designed to keep you at the top of your game.

* Successfully deployed with 75+ users on day one.

Bronx Analytical Project

As part of a comprehensive group project in my Python class, my team and I developed a dynamic webpage using GitHub, HTML, and SCSS to analyze four specific neighborhoods in the Bronx: Pelham, Melrose, Wakefield, and Belmont. Over five months, we collected and analyzed data on demographics, fuel usage, greenery, income, and building distribution. We leveraged multiple Python libraries, including matplotlib and numpy, to create highlighted maps and bar graphs illustrating the impact of carbon dioxide emissions in these neighborhoods. Using Folium and pandas, we designed interactive HTML maps that visualize this data, integrating specific information from CSV files. Additionally, I incorporated hyperlinks to individual pages, providing deeper insights into each dataset and analysis, making the project both comprehensive and user-friendly.
